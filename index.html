What is watchAny?

watchAny is a free, open-source media center built with Flutter that runs on Windows and Android. It plays virtually anything — local files, direct URLs, HLS streams, DASH streams, torrents via magnet links, and pulls anime and manga metadata from AniList.

Unlike traditional media players, watchAny combines a full anime/manga browser with a movie & series dashboard (powered by TMDB), a built-in torrent client (TorrServer), and AniList sync — all in one clean dark interface.

Key highlights

  • Hardware-accelerated playback via libmpv with 4K, HDR, and Dolby Vision support
  • Anime & manga dashboard with AniList integration and OAuth sync
  • Movies & series discovery via TMDB with Continue Watching history
  • Built-in TorrServer for magnet link playback without external apps
  • HStream support — stream from curated anime extension sources
  • Play any local file, URL, or stream by pasting it in the search bar
  • Zero ads, zero tracking, zero accounts required
💡 watchAny is completely free and open source. Download it from the GitHub releases page.

Platform Support

  • Windows 10/11 (x64) — full feature set, installer + portable ZIP available
  • Android 7.0+ (ARM64, ARMv7, x86_64) — split APKs for optimal size
ℹ️ macOS and Linux support is not officially available yet, but may work with Flutter builds from source.
