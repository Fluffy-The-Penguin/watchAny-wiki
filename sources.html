Usage

Media Sources

watchAny accepts virtually any media input — paste it in the search bar and hit play. Here's what's supported.

On this page
Direct URLs Torrents HLS / DASH Streams Local Files HStream Sources

🔗 Direct URLs

Paste any direct video URL into the search bar. Supported formats include:

  • MP4, MKV, AVI, MOV, WEBM, FLV (direct file links)
  • Any URL that resolves to a playable video stream
https://example.com/video.mp4 https://cdn.site.com/episode-01.mkv
ℹ️ Direct URL playback uses libmpv internally, which supports a huge range of protocols — HTTP, HTTPS, RTMP, and more.

🌊 Torrents (Magnet Links)

Paste a magnet link to stream directly via the built-in TorrServer engine. No external torrent client needed — watchAny streams while downloading.

magnet:?xt=urn:btih:HASH&dn=My+Movie+Name&tr=udp://...

TorrServer is launched automatically in the background when you paste a magnet link. The selected file begins streaming as soon as enough data is buffered.

  • Multi-file torrents show a file picker to choose which file to play
  • Downloaded pieces are cached locally for fast re-watching
  • The Downloads page tracks all active and completed torrents
⚠️ Only use torrent playback for content you own or that is freely and legally distributed. Respect your country's copyright laws.

📡 HLS & DASH Streams

Paste any HLS or DASH manifest URL to start live or VOD stream playback:

https://example.com/stream/index.m3u8 ← HLS https://example.com/stream/manifest.mpd ← DASH

These are used by IPTV providers, live TV services, and many anime streaming sources. Quality switching (adaptive bitrate) is handled automatically.

IPTV M3U Playlists

watchAny can load .m3u playlist files. Each channel in the playlist becomes a playable item.

https://example.com/channels.m3u

📁 Local Files

Open any local video file using the file picker (or drag & drop on Windows). Supported formats include everything libmpv handles:

  • MKV, MP4, AVI, MOV, WEBM, FLV, TS, M2TS
  • Any container format with H.264, H.265, AV1, VP9, etc.
  • Files with embedded subtitles and multiple audio tracks

External subtitle files (.srt, .ass) placed in the same folder as the video are automatically detected.

🔌 HStream Extension Sources

HStream lets you load Keiyoushi-compatible anime extension sources — JavaScript-based plugins that scrape anime streaming sites and return playable stream URLs directly in watchAny.

Setting up HStream

  1. Open Settings → HStream in watchAny
  2. Add a compatible source URL (Keiyoushi extension repo)
  3. The source appears in the anime detail page under "Stream" tab
  4. Select a source and episode to start streaming
💡 HStream runs extensions in an isolated JavaScript runtime — sources cannot access your system or personal data outside of watchAny.
