Getting Started

Installation

Get watchAny running in minutes. Choose your platform below — no sign-up, no payment, no bloatware.

On this page
Windows → Installer → Portable ZIP Android → Install APK Requirements First Launch

🪟 Windows

watchAny on Windows comes in two flavours — a standard installer and a portable ZIP. Both are identical in features.

Option 1 — Installer (Recommended)

Download watchany_setup.exe from the latest release and run it. The installer will:

  • Install watchAny to C:\Program Files\watchAny
  • Create a Start Menu shortcut
  • Register an uninstaller in Windows Settings
⚠️ Windows SmartScreen may show a warning since the app is unsigned. Click "More info" → "Run anyway" to proceed. The app is open-source and safe.

Option 2 — Portable ZIP

Download watchany_portable.zip, extract it anywhere (e.g. a USB drive or Desktop folder), and run watch_any.exe directly. No installation needed.

💡 The portable version stores settings alongside the EXE, so you can carry your configuration on a USB drive.

🤖 Android

Three APK variants are available. Pick the one matching your device architecture — if unsure, use arm64-v8a (covers most modern phones).

app-arm64-v8a-release.apk
Most Android phones since 2017 (64-bit ARM)
app-armeabi-v7a-release.apk
Older/budget 32-bit Android devices
app-x86_64-release.apk
Android emulators (BlueStacks, etc.)

Steps to install

  1. Download the correct APK from GitHub Releases
  2. Open your phone's Settings → Security → Install unknown apps
  3. Allow your browser or file manager to install APKs
  4. Tap the downloaded APK and press Install

System Requirements

🪟 Windows
  • Windows 10 or 11 (64-bit)
  • 4 GB RAM minimum
  • 200 MB disk space
  • GPU with DirectX 11 support
  • Internet connection for streaming
🤖 Android
  • Android 7.0 (Nougat) or higher
  • 2 GB RAM minimum
  • 150 MB storage
  • Internet connection for streaming

First Launch

On first launch, watchAny runs a Setup Wizard that lets you choose which sections to enable (Anime, Movies, Manga). You can change these at any time in Settings.

No account is required. To enable AniList sync (library sync, watch progress), you can optionally connect your AniList account from the Profile page.

ℹ️ watchAny will check for updates automatically on launch. You'll be notified if a new version is available.
